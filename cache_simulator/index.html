Multi-Level Cache Simulator

This is a college work (PUCPR - Brazil) from the discipline of Architecture and Computer Organization, 4th semester (2013). The objective was develop a simple multi-level cache simulator that have a specific architecture. The program receive some input values and calculates the cache miss and cache hit rates.

Developed by Aramis Hornung Moraes 2013
